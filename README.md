# Threejs-Portfolio
This is a personal portfolio website built using Three.js, a popular JavaScript library for creating 3D graphics in the browser. The portfolio showcases various projects and skills through interactive 3D elements.

## Demo

Check out the live demo of the portfolio [here](https://abhishek-ds-ml-gupta.github.io/Threejs-Portfolio/).

## Features
- Interactive 3D models and animations
- Responsive design for various devices
- Smooth navigation and transitions
- Integration with social media and contact forms
- Showcasing projects with detailed descriptions and images

## Technologies Used
- Three.js
- HTML5 & CSS3
- JavaScript (ES6+)
- WebGL

## Installation

1. Clone the repository: `git clone https://github.com/your-username/threejs-portfolio.git`

2. Navigate to the project directory: `cd threejs-portfolio`

3. Install dependencies: `npm install`

4. Start the development server: `npm run dev`

5. Open your browser and go to `http://localhost:5173` to view the portfolio.
